About The Book
===============

This book provides a practical introduction to machine learning with
applications to networking. The intended audience for the book are readers who
have some exposure or background in networking and are interested in learning
both practical concepts in machine learning and how these concepts can be
applied to a variety of problems in computer networking.

The book is accompanied by a collection of hands-on activities and exercises,
organized as course modules that track the organization of the book itself.
Each book chapter references these hands-on activities to allow the reader
both to understand the concepts in the abstract, as well as how to apply them
to networking problems in practice.
